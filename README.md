# Mississauga Vacuum Center Website

A complete static multi-page website designed for GitHub Pages or any basic web host.

## Key updates in this version
- Stronger homepage with business history, service detail, and trust messaging
- Serving since 1999 highlighted throughout the site
- Official Miele dealer and Miele warranty service messaging expanded
- All-brand sales, service, repairs, parts, and accessories messaging expanded
- Dedicated request-a-quote page
- Contact and quote forms now support photo uploads through FormSubmit once configured
- Central vacuum expertise, home visits, new builds, renovations, and retractable hose systems highlighted more clearly

## Important setup before launch
1. Edit `site-config.js`
2. Confirm the business email is set to `mississaugavacuum@gmail.com`
3. Upload all files to your GitHub repository root
4. Enable GitHub Pages

## Form notes
- The quote and contact forms are configured for a static website workflow using FormSubmit.
- Once the real email is added, submissions can be sent from GitHub Pages without custom backend code.
- File uploads are supported through the form setup in this version.
